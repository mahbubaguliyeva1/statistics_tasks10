Simulating a Counting Process with Rate λ

We simulate a uniform-in-time Bernoulli approximation of a Poisson counting process using n small subintervals of [0, T].

1. Simulation Framework

To approximate a continuous-time counting process, specifically a Poisson process with rate λ, we partition the observation window [0, T] with T = 1 into n equally sized subintervals. Within each subinterval we generate a Bernoulli random variable that indicates whether an event occurs. The event probability per subinterval is set to λ/n, ensuring that the overall expected number of events remains λ. This procedure is a classical discrete approximation to a Poisson process derived from the limit of rare events.

2. Limiting Process and Convergence Justification

The Bernoulli construction above converges, in distribution, to a Poisson process as n → ∞. Formally, if we let N_n(t) denote the cumulative number of Bernoulli events by time t, then the sequence of processes {N_n(t)} converges to {N(t)}, where {N(t)} is a Poisson process with rate λ. Specifically:

  • Convergence of finite-dimensional distributions: For each fixed t, N_n(t) converges to a Poisson(λt) random variable.
  • Independent increments emerge in the limit: The Bernoulli trials in disjoint intervals are independent, and their aggregated limits preserve this independence.
  • Stationary increments: Since each subinterval is treated uniformly, the limiting process inherits constant-rate increments.

Thus, the simulation is not merely heuristic—it is a discretization that reflects the formal rare-event derivation of the Poisson process.

3. Theoretical Properties of the Poisson Counting Process

Once the discrete simulation converges to its continuous counterpart, the resulting stochastic process satisfies the defining characteristics of a Poisson process. These include:

  • Distribution of counts: For any t ≥ 0, the number of events follows Poisson(λt).
  • Expectation and variance: E[N(t)] = Var[N(t)] = λt, reflecting equidispersion.
  • Sample-path behavior: Paths are almost surely right-continuous, integer-valued, and exhibit jumps of size one.
  • Interarrival times: Differences between successive event times follow Exponential(λ), a consequence of stationary and independent increments.

These properties make the Poisson process foundational in stochastic modeling, queueing theory, and applied probability.

4. Interpretation of the Rate Parameter λ

The rate parameter λ plays a central conceptual and mathematical role in the characterization of the Poisson process:

  • Intensity of events: It specifies the average number of events per unit time.
  • Temporal scaling: A higher λ implies denser event occurrences and shorter expected waiting times.
  • Link to exponential distribution: The mean interarrival time is 1/λ, establishing λ as the inverse-timescale governing the process.
  • Determinant of probabilistic variability: Since Var[N(t)] = λt, λ directly modulates the randomness inherent in the system.

In applications—ranging from photon arrivals in physics to call arrivals in telecommunications—λ embodies the fundamental rate at which stochastic events occur.
